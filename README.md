# node_server

## 关于
node全栈项目的后端部分，基于express，配合前端项目食用：[前端项目地址](https://github.com/EdisonChen-98/node_client)

## 技术栈
前端: vue 2 + element-ui + axios + ES6  
后端: nodejs + express + mysql  

## 模块
已有：登录模块，注册模块
在做：token权限，还有很多要做......

## 项目运行
```
git clone https://github.com/EdisonChen-98/node_server.git
npm install
nodemon ./app.js
```
